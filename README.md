# Countdown-Timer
- Here's a simple Countdown Timer made using HTML, simple properties of CSS and JS. 
- It's a countdown timer that counts down from a set time to zero, with options to stop, reset, and restart the timer.
## Technology Used
- HTML5
- CSS
- JS
